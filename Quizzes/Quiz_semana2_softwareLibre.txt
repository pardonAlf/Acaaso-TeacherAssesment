1. ¿Qué es el software?

A) Solo programas de internet
B) Parte física del computador
C) Conjunto de programas y datos
D) Solo el sistema operativo
E) Solo aplicaciones móviles

✅ Respuesta: C
💡 Explicación: El PPT define el software como el conjunto de programas, procedimientos, reglas y datos que hacen funcionar un sistema.

2. ¿Cómo se clasifica el software según sus libertades?

A) Básico y avanzado
B) Libre y privativo
C) Público y privado
D) Local y remoto
E) Antiguo y moderno

✅ Respuesta: B
💡 Explicación: Se divide en software libre y software privativo según las libertades que ofrece.

3. ¿Qué permite el software libre?

C) Usar, copiar, estudiar, modificar y redistribuir

E) Solo copiarlo

✅ Respuesta: C
💡 Explicación: Esa es la definición central del software libre según el PPT.

5. ¿Qué es un sistema operativo?
